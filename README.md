# Lettuces

> /!\ this is a hobby project and is not intended to use in production
> environments. Use at your own risk!

Lattice-based Cryptography in Rust

## Commands

### For development

- manually activate shell: `nix develop`
- run the usual cargo commands like: `cargo build/test/run`

### Packaging

- build: `nix build`
- run: `nix run`
